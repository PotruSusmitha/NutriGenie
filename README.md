# 🥗 NutriGenie

NutriGenie is an AI-powered nutrition recommendation system that generates personalized diabetes-friendly Indian recipes based on user health parameters. The project uses Large Language Models (LLMs) to recommend healthy meals while considering nutritional guidelines and user preferences.

---

## 🚀 Features

- Personalized meal recommendations
- Diabetes-friendly Indian recipes
- AI-powered recipe generation
- Health-aware dietary suggestions
- Easy-to-use Python application
- Nutrition-focused recommendations

---

## 🛠️ Tech Stack

- Python
- Hugging Face Transformers
- PyTorch
- GPT-2
- Pandas
- NumPy

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/PotruSusmitha/NutriGenie.git
```

Move into the project folder:

```bash
cd NutriGenie
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

---

## 📊 Dataset

The project uses an Indian food recipe dataset along with Glycemic Index (GI) values to generate healthier meal recommendations.

---

## 🤖 Model

The project uses a fine-tuned GPT-2 model for recipe generation.
